# 伪元素选择器

```html
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <title>伪元素选择器</title>
    <style>
        /*
            使用伪元素来表示元素中的一些特殊的位置

            :first-letter 首字母
            :first-line 首行

            :before 元素最前面的内容
            :after 元素最后面的内容

            一般这两个都需要结合content这个样式一起使用
            通过content可以向before或者after添加一些内容
        */
    </style>
</head>
<body>

</body>
</html>
```
