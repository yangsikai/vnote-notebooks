# 属性选择器

```html
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <title>属性选择器</title>
    <style>
        /*
            [属性名] 选取含有指定属性的元素
            [属性名=属性值]
            [属性名^=属性值] 以指定属性值开头的元素
            [属性名$=属性值] 以指定属性值结尾的元素
            [属性名*=属性值] 指定属性包含指定属性值的元素
            [属性名~=属性值] 有指定属性，且值为空格分割的列表，其中一个等于指定的属性值
            [属性名|=属性值] 有指定属性，且用-分割，以指定属性值开头，如果只有指定属性值也可以
         */
    </style>
</head>
<body>

</body>
</html>
```
