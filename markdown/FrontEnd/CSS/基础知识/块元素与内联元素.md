# 块元素与内联元素

div就是一个块元素：
- 块元素会独占一行的元素（无论它的内容有多少）
- div没有具体语义，只是一个纯粹的块元素，并且不会为它内部的元素设置任何默认样式

span是一个行内元素：
- 行内元素，只占据自身的大小，不会占用一行
- 多个行内元素可以在一行
- span也没有具体语义，是一个纯粹的行内元素
- span常用语给文字设置样式

    块元素主要用来在页面中布局
    内联元素主要给选中文本设置样式

    一般情况下不要用内联元素去包裹块元素
        例外：
            ！（a元素可以包含任意元素，除了a元素本身）
            ！p元素中不能包含其他任何块元素
